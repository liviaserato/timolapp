

## Preparando o projeto para transferência

A transferência de projetos no Lovable é simples e mantém todo o código e histórico. Sobre o Lovable Cloud (backend), há uma consideração importante:

### O que é preservado automaticamente
- Todo o código-fonte, componentes, páginas e assets
- Histórico de versões do projeto
- Configurações do projeto

### Sobre o Lovable Cloud
O Lovable Cloud **não pode ser desconectado** de um projeto uma vez adicionado. Porém, ao transferir o projeto, o Cloud acompanha — a nova conta terá acesso ao mesmo backend, banco de dados e edge functions.

### Passos para transferir

1. **Na conta atual (esta):**
   - Vá em **Settings** (clique no nome do projeto no canto superior esquerdo → Settings)
   - Na aba **General**, procure a opção **Transfer project**
   - Insira o e-mail da nova conta de estudante

2. **Na nova conta (estudante):**
   - Crie a conta em lovable.dev com seu e-mail de estudante
   - Aplique o desconto de estudante em https://lovable.dev/students
   - Aceite a transferência do projeto

3. **Após a transferência:**
   - O projeto aparecerá no workspace da nova conta
   - Todo o código, Cloud, banco de dados e edge functions continuarão funcionando
   - Se quiser, torne o projeto privado novamente nas configurações

### Importante
- Certifique-se de que a nova conta já existe antes de iniciar a transferência
- Se o projeto estiver conectado ao GitHub, pode ser necessário reconectar o repositório na nova conta
